
Coloca aquí las imágenes para tu tema de WordPress:

- logo.png - Logo principal de la web
- hero.jpg - Imagen principal de la página de inicio
- about-bg.jpg - Fondo de la sección Nosotros
- favicon.ico - Icono de favoritos
- Imágenes de servicios (service-1.jpg, service-2.jpg, etc.)
- Imágenes del equipo (team-1.jpg, team-2.jpg, etc.)

Es recomendable utilizar imágenes optimizadas para web (formatos .jpg o .webp para fotografías, 
.png para gráficos con transparencia) y con dimensiones adecuadas para evitar tiempos de carga lentos.
